# Machine Learning — Máster en IA & Data Science

Asignatura de algoritmos clásicos de Machine Learning con aplicación práctica en proyectos reales.

## Contenido

### Modelos Supervisados
- Regresión lineal, Ridge y Lasso
- Support Vector Machines (SVM)
- Árboles de decisión y Random Forests
- Gradient Boosting

### Aprendizaje No Supervisado
- K-Means y clustering jerárquico
- PCA y reducción de dimensionalidad
- Detección de anomalías

### Proyecto: Predicción de Miopía
Pipeline completo de ML para predecir la progresión de miopía en pacientes.
- Feature engineering sobre datos oftalmológicos
- Comparativa de modelos (SVM, RF, Gradient Boosting)
- Validación cruzada y optimización de hiperparámetros
- Datos: `X_train.csv`, `X_test.csv`, `Y_train.csv`
